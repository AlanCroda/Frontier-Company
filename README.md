# Frontier-Company

A default Unity project scaffold that can be added to Unity Hub and opened with Unity 2022.3 LTS or newer.

## Open in Unity Hub

1. Open Unity Hub.
2. Select **Add** / **Add project from disk**.
3. Choose this repository root (`Frontier-Company`).
4. Open the project. If your installed Editor version differs from `2022.3.62f1`, let Unity upgrade the project when prompted.

## Project layout

- `Assets/Scenes/SampleScene.unity` - default scene with a Main Camera and Directional Light.
- `Packages/manifest.json` - Unity package dependencies for a standard editor workflow.
- `ProjectSettings/ProjectVersion.txt` - Unity Editor version used for the project scaffold.
